# Write Getting Started Guide - Part 4: Simple Searches (Without SQL)

## Objective

Create Part 4 covering simple searches using OpenNotes CLI without requiring SQL knowledge. Focus on practical search patterns that beginners use daily.

---

## Steps

1. Introduce three types of searches:
   - Simple text search (`search` command)
   - Fuzzy matching (natural language similarity)
   - Boolean search (AND/OR/NOT operators)
2. Explain fuzzy search in beginner terms
3. Show example searches (finding by keyword, by fuzzy match, etc.)
4. Explain boolean operators (AND, OR, NOT)
5. Show combining search methods
6. Add troubleshooting (why search didn't find something)
7. Include 5-6 copy-paste examples
8. Link to SQL guide for advanced searching

---

## Expected Outcome

Complete Part 4 (~700 words) with:
- ✅ Three search methods clearly explained
- ✅ No SQL terminology (beginner friendly)
- ✅ Real-world search examples
- ✅ Boolean operators explained simply
- ✅ 5-6 working copy-paste examples
- ✅ Common search mistakes
- ✅ When to use each search type
- ✅ Graduation path to SQL
